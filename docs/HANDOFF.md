## ЧТО ТАКОЕ ЭТА ИГРА
- Мотоцикл-силуэт мчит по бесконечному процедурному **свечному графику** (японские свечи: зелёные/циан = рост, красные = падение, с фитилями).
- Управление: удержание = газ; отпускание = наклон носа вниз; в воздухе удержание = сальто назад (бонус-монеты за флипы).
- Крах-приземление под кривым углом = **LIQUIDATED** (геймовер), дистанция/рекорд, монеты вдоль трассы.
- Атмосфера трейдинг-терминала: ценовая шкала справа ($-уровни), тонкая сетка, живой тикер «BTC $цена» (привязан к высоте байка), индикатор настроения (bull run / crash / sideways).
- Визуал: градиентное сумрачное небо (смена времени суток), параллакс-хребты, неоновый шлейф за байком, частицы, виньетка.

## ПОЧЕМУ ИМЕННО ЭТО НАПРАВЛЕНИЕ (сошлось по всем осям, в которые бились)
1. **Тема в корень проекта** — cryptobill = крипта; графики цен это ядро, а не натянутый сюжет.
2. **Уникально + есть тяга** — вдохновлено StonkRider (виральный на Reddit: езда по биржевому графику; https://stonkrider.world). Но он ПЛОСКИЙ и без глубины — наш шанс сделать богаче.
3. **Решает наш визуальный тупик** — линия графика + силуэт = «красота из ограничения». Alto's Odyssey доказал: простая форма + великолепный свет/градиент/частицы = Apple Design Award, и это код может БЕЗ художника.
4. **Механика проверена хитами** — Hill Climb Racing (100M+), Gravity Defied (культ), Alto (награды).
Как бить StonkRider: роуглайт-прогрессия, разные байки/«трейдеры» (= мечта пользователя о разных персонажах), премиум Alto-визуал, крипто-лор (ликвидация, памп/дамп, Freedom-vs-Control).

## ЭКСПЕРТИЗА + ГЕЙМ-ДИЗАЙН (сессия 2026-06-14, по просьбе «сделай экспертизу как продюсер»)
Диагноз: ехать было НЕ за чем — гемы ни на что, только рекорд дистанции; нет скилл/комбо; нет кривой сложности; краш необъясним; крипто-тема только декор; джус наполовину; нет обрамления забега.
**P0 СДЕЛАНО (чистая механика, физику не трогал):**
- **LEVERAGE (множитель)** `mult`: растёт от воздуха (+0.01/f), скорости>11 (+0.0035/f), сальто (+0.6×flips), near-miss (+0.5); падает при vx<6; кап 12. Бейдж ×N под HUD (cyan→green, пульс).
- **PnL ($)** `pnl`: += vx×mult×0.05/кадр + гемы (25×mult) + сальто (40×flips×mult) + near-miss (30×mult). Рекорд `bestPnl` в localStorage `cipher_best_pnl`. PnL — герой-число HUD; экран смерти = сводка PnL + peak ×N + NEW BEST.
- **Кривая сложности**: `pickRegime` берёт `d=dist/1800`, растит волатильность и долю pump/crash.
- **Джус**: near-miss «+$» (skim<15px при airT>16), всплывашки `floats`/`floatPop`, hit-stop на сальто (4f)/near-miss (2f), зелёная вспышка `gflash`, пульс множителя.
**P1 СДЕЛАНО (сессия 2, «доведи до финала, думай как продюсер/маркетолог/игрок»):** игра из демо стала ЦЕЛЬНОЙ:
- **State-машина** `screen`: 'title'|'play'|'paused'|'dead'. `toTitle()` фреймит байк по центру. Ввод переписан в `pointerDown` с тап-зонами (`UI.*` rect-ы, `hit()`), клавиши: пробел газ/старт, Esc/P пауза, M мьют, R рестарт.
- **Титул** «CHART RUNNER» + слоган + BEST $PnL + банк ◆ + живой байк-герой + ‹ › выбор + ПОЕХАЛИ + список 3 целей.
- **Байки/«трейдеры»** `BIKES[]` (HODLER/DEGEN/WHALE/SATOSHI): палитра (body/rim/accent/rider) параметризована через `BK()`; разблок за гемы из `bank` (localStorage `cipher_bank`/`cipher_owned`/`cipher_bike`).
