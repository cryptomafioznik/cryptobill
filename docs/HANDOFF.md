1. **Тема в корень проекта** — cryptobill = крипта; графики цен это ядро, а не натянутый сюжет.

- Запустить preview-сервер (preview_start "cryptobill"), затем открыть в браузере пользователю: `open "http://localhost:4173/toys/chartrider.html"` (через Bash с dangerouslyDisableSandbox).
- На телефоне (та же сеть): `python3 -m http.server 4173 --bind 0.0.0.0` из корня проекта → `http://192.168.1.128:4173/toys/chartrider.html`.
- Для проверки геймплея в preview без рук: физика `update()` и состояние (`bike`, `terr`, `screen`, `down`) доступны в глобале — можно прогнать заездом через preview_eval (down=true в цикле update), затем preview_screenshot. ВАЖНО: `down` не «прилипает» под синтетическим вводом preview — гнать update() вручную в цикле.
