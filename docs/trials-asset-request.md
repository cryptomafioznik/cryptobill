# CRYPTOBILL trials25d — ПОЛНЫЙ запрос ассетов (для генерации одним пакетом)

Цель: articulated-байк (катящиеся колёса + подвеска + плавные позы), материал трассы, бесшовный скролл-фон. Всё, что НЕ здесь (частицы, тень, искры, speed-lines, glow, неон-края трассы, HUD-кнопки) — делается кодом, ассеты не нужны.

## Общие правила (важно для сборки)
- PNG, **прозрачный фон**, объект **строго по центру** кадра, без тени на земле, без рамки/подписи.
- Единый стиль: синтвейв-неон, глубокие фиолет/магента/циан, чистая детальная иллюстрация, лёгкое свечение.
- **Один и тот же дизайн байка и райдера во ВСЕХ файлах** (бело-красный мотокросс-байк, белый шлем, бело-красная экипировка). К каждому промпту прикладывать `bike_01_neutral_ground.png` как эталон стиля.
- Байк во всех позах — **вид сбоку с лёгким ¾ (нос вправо), ОДНОГО размера и по центру** (чтобы части и позы совмещались).
- Колёса — **идеально круглые, ступица точно в центре кадра** (я их вращаю; смещённая ступица = кривое вращение).
- Тайл-текстуры и слои фона — **бесшовные по горизонтали** (левый край стыкуется с правым).

## Группа 1 — БАЙК (articulated), 1024×1024
1. `bike_body_neutral` — рама+бак+сиденье+руль+передняя вилка+задний маятник+**райдер** (нейтральная посадка), **БЕЗ КОЛЁС** (на месте колёс прозрачно; концы вилки и маятника чётко видны).
2. `bike_body_accel` — то же, райдер наклонён **вперёд** (разгон), без колёс.
3. `bike_body_wheelie` — то же, вес **назад**, перёд приподнят, без колёс.
4. `bike_body_brake` — то же, **нос вниз** (торможение), без колёс.
5. `bike_body_air` — то же, **в воздухе**, поджато, без колёс.
6. `bike_body_land` — то же, **жёсткая посадка** (подвеска сжата, райдер присел), без колёс.
7. `wheel_front` — только переднее колесо, круглое, ступица в центре, внедорожный протектор, неон-обод.
8. `wheel_rear` — только заднее колесо (звезда/диск), круглое, ступица в центре, чуть крупнее, неон-обод.
9. `crash_bike` — упавший байк на боку (с колёсами).
10. `crash_rider` — райдер отдельно (кувырок/лежит, без крови/травм).

## Группа 2 — ТРАССА (бесшовные тайлы)
11. `track_top_tile` (512×128) — БЕСШОВНАЯ горизонтальная полоса верхней поверхности трассы: тёмный неон-техно-асфальт, тонкая сетка/панели, лёгкое свечение.
12. `track_side_tile` (512×96) — БЕСШОВНАЯ полоса боковой грани (темнее верха, тех-панели).

## Группа 3 — ОКРУЖЕНИЕ (бесшовные слои для прокрутки)
13. `env_sky` (1024×1024, бесшовно по бокам) — синтвейв-небо (фиолет→магента→оранжевый низ) + очень бледные свечи-график + сетка. **БЕЗ солнца.**
14. `sun` (700×700, прозрачный) — большое ретро-солнце с горизонтальными полосами + мягкое свечение, отдельно.
15. `city_far` (1024×700, бесшовно, верх прозрачный) — дальний скайлайн (силуэты, тусклые окна).
16. `city_mid` (1024×760, бесшовно, верх прозрачный) — средний город (ближе, ярче окна).
17. `palms_fg` (1024×520, бесшовно, верх прозрачный) — передний план: тёмные пальмы/листва вдоль низа.

## Группа 4 — ОБЪЕКТЫ
Можно оставить из прошлого пакета (они ок) ИЛИ перегенерить в этом же стиле: монета-биткоин, токен, энергия, канистра-буст, ящик-препятствие, флаг-чекпоинт, зона-приземления, предупреждение-ликвидация.
